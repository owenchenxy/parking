## 测试用例
**注意，请在tests目录下运行测试代码。**

### 情景一
**Given**: 三辆车都在第1个和第2个停车场中间，三个停车场都还有2个位置
**When**: 执行调度
**Then**: 距离第2个停车场最近的car3/car2被调度到第2个停车场，car1被调度到第3个停车场
![Alt Text](../images/case1.png)
运行以下代码测试：
```
./clean.sh

python3 test_1.py

kubectl get pod -o wide
```
